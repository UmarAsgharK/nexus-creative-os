# Quick Start Checklist

Use this checklist to get your project on GitHub and deployed quickly.

### 1. GitHub Setup (5 minutes)

```bash
# In your project directory
git init
git add .
git commit -m "Initial commit: Modern animated landing page"

# Create repository on GitHub.com, then:
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main
```

### 2. Deploy to Vercel (3 minutes)

1. Go to [vercel.com](https://vercel.com) â†’ Sign up with GitHub
2. Click "Add New Project"
3. Import your repository
4. Click "Deploy" (settings auto-detected)
5. Copy your live URL

### 3. Update README (2 minutes)

1. Open `README.md`
2. Replace `https://your-deployment-url.vercel.app` with your Vercel URL
3. Update author information (name, links)
4. Commit and push:
   ```bash
   git add README.md
   git commit -m "Update README with live demo link"
   git push
   ```

### 4. Add Screenshots (Optional but Recommended)

1. Take screenshots of your site
2. Create `screenshots` folder
3. Add desktop.png, mobile.png, animations.gif
4. Update README screenshot paths

### 5. Add to Portfolio

- Add project to your portfolio website
- Use the description from DEPLOYMENT_GUIDE.md
- Link to both GitHub and live demo
